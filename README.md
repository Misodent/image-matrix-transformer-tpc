# Dotfiles
───────────────────────────────────────────────  
Dionysus vers. 1.1
───────────────────────────────────────────────  

Rice config for **Hyprland** on Arch Linux,
running on my **ThinkPad X1 Carbon** (_dionysus_)

## Features
  - Animated **Neofetch**
  - Dynamic **Waybar**
  - ASCII **Cava Visualiser**
  - Nord-inspired **neon-radioactive theme**
